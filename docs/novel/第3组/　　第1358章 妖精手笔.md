# 　　第1358章 妖精手笔
　　沙之舟跑掉了，如果张明杰的罪名落实了，虎姐也不会是刚才那种反应了，所以，类似的事情，谁敢保证不会再一次发生呢？我心中苦笑，可望着楚缘写满了紧张与恐惧的一张小脸，我实在不忍她也像我这般悲观，握住她颤抖而冰凉的一只小手，柔声道：“嗯，哥答应你，这种事情，这辈子都不会再发生了。”
　　楚缘使劲点头，仿佛我说的话会成为世界运转的剧本似的，百分百不会错，释然之后，又笑了，“哥你没听懂我的意思，我不是说你凶波波姐凶得不对，毕竟她是有前科的，再胆大妄为的事情，她既然敢说，那就一定敢做，就连东方也跟我说，波波姐好像胆子很小，但只是怕疼，并不是怕事，其实胆子比她大得多，因为她只是手狠，波波姐却是心狠，所以她敢做的，波波姐一定不敢做，但波波姐敢做的，却多半是她想都不敢去想的。就说上次咱们在度假村出了事，东方虽然激动，可没有为此冤枉任何人，但波波姐明明知道那个江玉不是主使者，还是硬要迁怒他，要不是咱们及时脱险，下一个掉下山的，肯定就是江玉了，这次也是，虽然这几天程流苏总是拜托波波姐照顾我，可私下里却是对我千叮万嘱，要我看住她的……”
　　“看住她？”我微微蹙眉，“她又想做什么？”
　　楚缘叹了口气，道：“小夜姐姐嫌丢人，又怕你生气，肯定没敢告诉你——从你出事以后，波波姐就让桑叔叔带了百十号人，天天吃住在医院里，警察拿他们都没有半点办法……”
　　“什么？！”我气得伤口差点炸开。
　　“你先别急，听我说完嘛，”楚缘先安抚了我，然后才慢慢道：“就算小夜姐姐不说，明天还是会有人告波波姐的状，倒不如我先跟你说了，免得待会波波姐向你认错，瞒也不是，不瞒也不是，少不了又挨你一顿骂——她的初衷是好的，既为保护你，也为看住张明杰，就像三小姐姐姐信不过任何人，波波姐也不相信任何人了；但她的脾气是很不好的，幸亏你今天醒过来了，不然再昏迷个两三天，可真是谁也说不好了，波波姐又会做出什么惊人的事情来……”
　　“又？”我一阵头疼脑大，“你这么说，就是她已经做出了什么惊人的事情啊……”
　　“嗯，”楚缘愁笑道：“桑叔叔的人都是装病住进来的，张明杰住在六楼，所以他们就金钱开道，软硬兼施，清空了余下的所有病房，也全部住进了六楼，开始那两天还好，只是盯人，倒也安分，可……哎，张明杰一直抵赖，说自己也是受害者，不承认是他主使沙之舟伤你，警方又找不到太有力的证据，波波姐的火气就越来越大了，和张家人摩擦不断，昨天上午，终于闹出事了……”
　　想到妖精刚才说，她若出手一准儿比东方过分，我这心脏就有点受不了，“他把张明杰怎么了？”
　　“警方没有证据，但也不信张明杰无辜，以保护为名把他给监控起来了，波波姐倒是没能把他给怎么样呢，倒了霉的，是张明杰那个嘴欠的堂弟张培文，”楚缘又是气恼又是无奈，道：“张培文颠倒黑白不说，还天天咒你，确实让人忍无可忍，可波波姐也实在是让人防不胜防——昨天柳晓笙和星雨姐姐来探病，星雨姐姐看到你这惨样儿，哭惨了，波波姐就把她拉到客房去了，我以为她是安慰星雨姐姐呢，大意了，没去听她和星雨姐姐说了什么，结果，柳晓笙告辞，说下楼再探望一下张明杰，可没多一会，就听说张培文被人从楼梯给踢下去了，人肉皮球似的，你一脚我一脚的往下踹，从六楼一路滚到二楼，尾椎骨骨折，右小臂还是右小臂来着，骨折，两条小腿胫骨骨折，肋骨断了三根，说是滚楼梯摔的，其实八成都是人为伤，最惨的就是那张脸，又挨踢又挨撞的，连个人模样都看不出来了，肿得像猪头，再加上一身皮肉伤，瞧着比哥你还惨呢。张家人怎肯罢休？当时就报了警，波波姐就学他们一样抵赖，非说张培文是看到星雨姐姐太激动了，追她的时候自己不小心失足从楼梯上摔下去的，根本没人踢他，大家都是病人，身体弱手脚慢，最多就是没能及时的拉一下扶一把，还有人故意躲开了，难道没爱心也有罪吗？那他们还要告张培文哩，拐着弯的往楼下滚，不是蓄意伤人，就是想碰瓷讹人！可是哥你说，星雨姐姐那么讨厌张培文，为什么把他约出病房啊？如果不是波波姐事先安排好的，哪有那么巧，同一天入院的百十个没病的病人会在同一时间排着队的爬同一条楼梯？张家人要监控录像，结果也不知道是萧叔叔还是三小姐姐姐摆平了院方，硬是说昨天上午恰好监控设备检修，所以录像滴没有，警方对张家的怨气现在比谁都重，连和稀泥都算不上了，除了姓张的说什么都他们不信之外，其他人说什么他们都信……”
　　我气得说不出话来，也怕一张口就忍不住笑出来——妖精的做法百分百不对，却学足了张家人的睁眼说瞎话，颇有些以彼之道还施彼身的风格，不但会点火，她还会扇风。
　　果然是只妖精啊……
　　楚缘继续道：“张家人为这事闹了一天，大概也知道吵不出结果，到了晚上就偃旗息鼓了，然后市局那个林叔叔过来和小夜姐姐漏了口风，说张家人今天会安排张明杰和张培文转院，他无权拦着，虽然不知道今天为什么没了动静，可当时一看波波姐气炸了毛似的，摆明了今儿还要给张家人更厉害的颜色看看，我就赶紧偷偷给东方打了电话，让她一早过来，和她演了整整一天的戏，你看我脸哭得花，那是因为一天都没敢擦，波波姐怕我和东方在你面前吵架，哪还有心思去找张家人的麻烦？”
　　我好奇道：“你和东方不是正在斗气吗，她怎么还愿意陪你演戏啊？”
　　楚缘翻我一记白眼，道：“你以为东方和你一样不识好人心啊？我不让她伤人是为她好，她一时半会生我的气，但平静下来，就向我道歉了，我俩一直假装斗气，就是怕波波姐以为我俩没事了，就放心出去找张家人的事了。”
　　我不禁欣慰的笑了，抬手想抚楚缘的头，臭丫头怕我动作大了牵痛伤口，紧忙离开软凳，蹲跪在床边，将脑瓜顶送到我手掌下边，一副很得意很享受的傻模样。
　　“刚才看你和一可一样生气，我还觉得你们俩都没长大，现在再看，原来你已经是个懂事的大姑娘了。”
　　楚缘却扁扁嘴，道：“我是大姑娘了，但还不够懂事，我自己是知道的……哥，我和东方假装吵架，是程流……是流苏姐姐教我们的，她说除了你，波波姐是谁也哄不了劝不住的，我不行，东方不行，就是萧叔叔也不行，所以她才拜托波波姐照顾我，说哄我劝我，是现在唯一能让波波姐分心的事情，这样她就不会一门心思去找张家人的麻烦了，流苏姐姐说的话，与你刚才说波波姐的话差不多，她也觉得波波姐用这样的方式即使制裁了张明杰，可是弄脏了自己的手，赔上了自己的前途，你非但不会开心，还会很伤心……”
　　知我者，程姑奶奶也……我是喜忧参半啊——程姑奶奶知道如何制约萧妖精，岂不证明，她十分了解萧妖精对我的感情？
